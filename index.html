Como preparar o Google Sheets

Para o HTML conseguir ler os dados, publique a planilha ou deixe o arquivo acessível para quem tem o link.

1. Crie 3 abas
Use nomes como Estoque, Movimentação e Financeiro.
2. Mantenha os cabeçalhos
Os nomes das colunas precisam bater com os campos listados ao lado.
3. Para planilhas simples
Cole os links das abas com gid. O dashboard tentará ler direto.
4. Para planilhas com fórmulas
Use a API do Google Apps Script. Ela lê os valores exibidos na tela com getDisplayValues().
